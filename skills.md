# Skills

A showcase of skills and tools built by participants.

| Project | Description | GitHub |
|---------|-------------|--------|
| **Sci-Brain** | A skill to mimic the question patterns of great scientists, leading new researchers to think deeper. | [QuantumBFS/sci-brain](https://github.com/QuantumBFS/sci-brain) |
| **Auto Coder-Tester** | A skill that helps the agent understand your intent, ask clarifying questions, plan with detailed explanations, and automatically code with built-in testing. | [Lizt1996/CC-Skill-Auto-Coder-Tester](https://github.com/Lizt1996/CC-Skill-Auto-Coder-Tester) |
| **Boltzmann Collision Integral** | A theoretical and numerical study of the linearized Boltzmann collision operator for electron-electron scattering, focused on transport calculations beyond the relaxation-time approximation. | [HongYuan129/boltzmann_transport](https://github.com/HongYuan129/boltzmann_transport) |
| **GMX-Concierge** | A skill that guides you through preparing GROMACS molecular dynamics simulations, automating the routine yet error-prone setup process. | [DivingWhale/GMX-Concierge](https://github.com/DivingWhale/GMX-Concierge) |
| **Moire Tight-Binding** | Build effective tight-binding models for moire systems — from first-principles DFT on HPC to Slater-Koster fitting and physical property computation. | [47Wu/hackathon-2026](https://github.com/47Wu/hackathon-2026) |
| **PolyPhys Agent** | An agentic skill suite for theoretical biophysicists to explore the relationship between microscopic physical parameters and macroscopic structural behavior in chromosome polymer modeling. | [CodePioneer42/PolyPhys](https://github.com/CodePioneer42/PolyPhys) |
| **easy-ssh** | Local AI, Remote Execution — a Bash CLI tool that bridges local AI-assisted development with running compute-heavy jobs on remote servers. | [exAClior/easy-ssh](https://github.com/exAClior/easy-ssh) |
| **Design Mott Materials** | Use exact diagonalization with second-order perturbation to obtain effective models for strongly correlated insulating materials with active d or f electrons, based on ab initio calculations. | [SQNing/Design_Mott_Materials](https://github.com/SQNing/Design_Mott_Materials) |
| **Condensed Matter Paper Reproduction** | Reproduce calculations from condensed matter physics theory papers, generating a markdown plan of physical principles and procedures before coding in Python. | [JohnWatson2333/condensed-matter-paper-reproduction-skill](https://github.com/JohnWatson2333/condensed-matter-paper-reproduction-skill) |
| **Wigner Crystal Ansatz** | Construct suitable ansatz trial density matrices that converge under Hartree-Fock self-consistent iteration to Wigner crystal states with different lattice structures. | [hezc1928/wigner_crystal](https://github.com/hezc1928/wigner_crystal) |
| **Paperer** | Search for high-quality articles, summarize them with a fixed structure, and organize papers within a single HTML page by topic and time. | [QiushanHuang/paperer-skill-plugin](https://github.com/QiushanHuang/paperer-skill-plugin) |
| **Arpes_Ana** | Load ARPES data and extract EDC and MDC curves for analysis. | [KeyuAN95/Arpes_ANA](https://github.com/KeyuAN95/Arpes_ANA) |
| **SlidesMentor** | Teaching-First NotebookLM — study a research paper with its code, understand the core ideas, and turn the material into clear teaching content like lecture slides. | [QingyunQian/SlidesMentor](https://github.com/QingyunQian/SlidesMentor) |
| **PaperDojo** | A thinking coach built on daily paper feeds — do a 30-min thinking exercise, see the problem first, try to solve it with an AI coach, then compare with the authors' approach. | [HsiaHongKuan/paperdojo](https://github.com/HsiaHongKuan/paperdojo) |
| **Quill** | Give it a topic and it teaches you the field through a four-act failure-driven narrative: the problem, the failed attempts, the solution landscape, and the open frontier. | [Br0kenSmi1e/quill](https://github.com/Br0kenSmi1e/quill) |
