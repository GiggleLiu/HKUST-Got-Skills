# HKUST-Got-Skills

**Design a skill to automate and accelerate scientific discovery.**

A cross-campus hackathon at HKUST(GZ) and HKUST(CWB) where teams build AI coding skills for scientific workflows.

**Website**: https://giggleliu.github.io/HKUST-Got-Skills/

## What Are "Skills"?

Skills are reusable automation modules for AI coding tools (Claude Code, Cursor, Codex CLI, OpenCode, etc.) that help scientists work faster — from literature reviews to benchmark generation to paper writing.

## Local Development

```bash
make install   # Install Jekyll and dependencies
make serve     # Serve site locally with live reload
```
